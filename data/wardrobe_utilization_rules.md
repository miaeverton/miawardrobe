# WARDROBE UTILIZATION RULES
## Maximizing the value of every item

---

## THE GOAL

The goal is not only to create great outfits. It is to maximize the value of the full wardrobe. Mia should wear more of what she owns, not repeatedly reach for the same 15 pieces.

---

## RULES

1. At least **25% of outfit recommendations** must incorporate pieces that appear infrequently in previous recommendations.
2. When presenting multiple outfit options, **never repeat the same shoe, bag, or outerwear** across options in the same session.
3. **Do not anchor every outfit** around the same hero pieces — see overused list below.
4. **Surface forgotten pieces** regularly. When a recommendation is being finalized, ask: is there an underused piece that could work here?
5. For every outfit, **label each piece** as: Hero / Supporting / Underused.
