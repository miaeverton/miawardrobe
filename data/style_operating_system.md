# STYLE OPERATING SYSTEM
## Mia's Outfits Project — Master Reference

*This document governs every outfit recommendation in this project. Read it before building anything.*

---

## STYLE MISSION

Dress like a confident, sophisticated, creative woman with exceptional taste.

The goal is NOT to look fashionable, trendy, flashy, or overly styled.
The goal IS to look effortlessly put together, design-conscious, intelligent, and interesting.

Every outfit should communicate:
- Confidence
- Ease
- Sophistication
- Creativity
- Worldliness
- Exceptional taste

People should think:
- "She always looks good."
- "She has great taste."
- "That outfit feels effortless."
- "She knows who she is."
- "Everything she wears feels intentional."

NOT: trendy / influencer-like / flashy / overly feminine / corporate / over-accessorized / fashion victim.

## BODY CONSIDERATIONS

**Height:** Tall
**Build:** Athletic, hourglass

**What this means for styling:**
- Flat shoes do not shorten — height absorbs them
- Wide-leg trousers mirror shoulder width, creating a balanced A-frame
- Maxi lengths work — the frame carries them without being overwhelmed
- Cropped outerwear shortens the torso and interrupts the long line — avoid
- Long coats and dusters read as elegant, not overwhelming
- Chunky soles and ankle straps interrupt the vertical line — avoid

**Prioritize:** waist definition, strong proportions, long lines, structured outerwear, wide-leg trousers, barrel pants, clean necklines.

**Avoid:** excessive volume everywhere simultaneously, overly complicated details, clothing that hides shape entirely.

## OUTFIT PHILOSOPHY

Every outfit must have:
1. One interesting silhouette
2. One grounding neutral
3. One intentional styling choice (unexpected footwear / architectural shape / interesting proportion / layering element / statement texture)

Do NOT rely on bright color or trend to create interest.
Use silhouette, proportion, texture, fabric, and styling first.

---

## STYLING PRIORITY ORDER

1. Proportion
2. Silhouette
3. Fabric
4. Texture
5. Styling details
6. Color
7. Trend relevance ← always last

## COLOR STRATEGY

**Core:** cream, ivory, camel, chocolate, navy, black, denim, olive
**Accents:** burgundy, rust, forest green, soft blue

Color supports the outfit. It is never the primary source of interest.
One color accent at a time. Never two simultaneously.

## TUCK FRAMEWORK

Every outfit recommendation must include exactly one tuck instruction, stated plainly. Never present it as a choice.

- **Full tuck** — fully in, clean waistband visible. For: ribbed tanks, fitted tees, silk camis, anything lightweight. Best against high-waist silk trousers, dark indigo jeans, any beautiful waistband.
- **Half tuck** — front only, two inches, centered. Back stays out. For: relaxed or slightly oversized tops over casual bottoms. Must be precise or it reads sloppy.
- **Untucked** — completely out. For: cropped tops that end naturally at or above the waistband. Also for chunky knits — pull slightly forward to blouse at front. Never untuck a long top over wide-leg trousers without a belt.
- **Open layer, inner tucked** — outer piece worn fully open and left out. Inner piece always fully tucked. Belt optional.
- **Bloused** — tucked in but pulled out slightly to create a soft fold. For silk or satin skirts where a full tuck reads too stiff.

**Rules:**
- Fitted tops → almost always full tuck
- Cropped tops → never tucked
- Oversized/relaxed tops over wide-leg bottoms → half tuck or belt, never fully untucked without one
- Blazer over a top → top beneath is fully tucked
- Dresses → no tuck instruction needed; note if a belt is/isn't appropriate

## ALISON BORNSTEIN FOOTWEAR FRAMEWORK

Apply to every outfit:
1. What does this shoe do to the leg line? (Pointed toe elongates. Round toe shortens. Chunky sole adds weight at base.)
2. Does the shoe balance the outfit's weight? (Heavy/voluminous top → sleek minimal shoe. Minimal outfit → interesting shoe creates the moment.)
3. Is there a wrong-shoe move available?
4. Always identify primary shoe + one alternative. Explain which is stronger and why.
5. Flat shoes are a choice, not a compromise. Justify them as choices.

## ANTI-MILLENNIAL CHECKLIST

Run before finalizing any recommendation. These combinations must be avoided:

- **Ruffled/feminine blouse fully tucked into high-waist jeans with a coordinated block heel** — the millennial Instagram uniform. If a feminine/ruffled top is present, disrupt it: unexpected shoe, open layer, or colour contrast.
- **Over-coordinated accessories** — matching bag to shoe exactly, matching belt to bag. Use tonal coherence, not literal matching.
- **Statement accessory added to rescue an unresolved outfit** — the interesting element must come from the combination of pieces, not a single accessory applied to a boring base.
- **The "dressed up jeans" move** — light wash jeans + silk blouse + structured bag + heels = trying too hard. Keep one element deliberately casual so the jeans feel chosen.
- **Long cardigan as an outer layer over a complete outfit** — reads formless, eliminates the silhouette. Use shirts and blazers as layers, not cardigans.
- **Midi skirt + chunky ankle boot** — millennial-boho. This wardrobe has no chunky ankle boots; keep it that way in recommendations.

**The test:** Does this look like it was assembled to look stylish? Or does it look like it was assembled by someone who has taste and didn't think about it that hard? Always the second one.

## PURCHASING RULES

Before recommending any purchase:
1. Does it support existing outfit formulas?
2. Does it work with at least 5 items already owned?
3. Does it strengthen the style identity?
4. Is it versatile?
5. Is it better than what already exists?

If not → recommend against buying.
